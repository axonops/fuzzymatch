# Code of Conduct

This project adopts the **Contributor Covenant, version 2.1**, in
full. The complete text is published at:

- <https://www.contributor-covenant.org/version/2/1/code_of_conduct.html>
- Plain text: <https://www.contributor-covenant.org/version/2/1/code_of_conduct.txt>

By participating in this project — opening issues, opening PRs,
posting comments, attending project events — you agree to be bound by
that Code in its entirety.

## Reporting

Reports of Code of Conduct violations are received at:

```text
conduct@axonops.com
```

Reports are handled confidentially by the AxonOps maintainer team.
Reporters will receive an acknowledgement within 2 business days and
a substantive response within 7 business days.

Where the report concerns a maintainer, the matter is escalated to
AxonOps leadership outside the project.

## Scope

The Code of Conduct applies in every project space — the GitHub
repository (issues, PRs, discussions, code comments), the project's
Slack/Discord channels if and when they exist, in-person and remote
project events, and any public space where a contributor is
representing the project.
