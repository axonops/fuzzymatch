## Sampling Rate

- **After every task commit:** Run `go test ./...` (quick suite)
- **After every plan wave:** Run `make check` (full quality gate including coverage Floor 1/2/3, bench-diff, determinism golden)
- **After Q11c (DM paper-anchored test) lands:** Run the test in isolation BEFORE allowing Q9 to start. Gap 6 dependency.
- **After Q10 (cross-validation corpora) lands:** Run `make regen-cross-validation && go test ./...` and confirm `vectors.json` is committed with the loader test reading it.
- **After Q11b (FMA double-cast) lands:** Run cross-platform determinism golden (`make verify-determinism`) on the full CI matrix — linux/amd64, linux/arm64, darwin/amd64, darwin/arm64, windows/amd64.
- **After Q7a (DM `[4]byte` optimisation) lands:** Run `go test -bench=BenchmarkDoubleMetaphone -benchmem -count=10` and confirm allocs drop from 6/op to ≤ 2/op.
- **After Q8b (Tokenise ASCII fast path) lands:** Run `go test -bench=BenchmarkTokenise -benchmem -count=10` AND `go test -bench=BenchmarkTokenSortRatio -benchmem -count=10` (token-tier inheritance) and confirm allocs drop per Q8b budget.
- **Before `/gsd-verify-phase 8.5` verification gate:** Full suite must be green AND re-run the Phase 8 comprehensive review prompt with 14 specialist agents → assert zero Critical + zero Important findings.
- **Max feedback latency:** 30 s quick / 5 min full per wave.

## Cross-Platform Determinism Gate (load-bearing for Q11b)

Q11b (FMA-defeating double-cast at `cosine.go:343` and `scorer.go:380`) is the most platform-sensitive change in Phase 8.5. The CI matrix (`linux/amd64`, `linux/arm64`, `darwin/amd64`, `darwin/arm64`, `windows/amd64`) MUST produce byte-identical `testdata/golden/algorithms.json` and `testdata/golden/scorer-default.json` after the change. Verification commands:

```bash
make verify-determinism    # local single-platform sanity
# CI matrix runs the actual five-platform diff on every PR
```

If any platform diverges post-Q11b, the Phase 8.5 plan must NOT proceed to subsequent tasks until the divergence is resolved (rollback Q11b on that platform, document the deviation, escalate).
