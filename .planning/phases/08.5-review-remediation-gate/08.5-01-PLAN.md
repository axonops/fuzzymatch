| Boundary | Description |
|----------|-------------|
| caller → fuzzymatch public API | Untrusted parameter inputs may attempt to discriminate sentinels via string matching (S — Spoofing) or trigger panics whose values cannot be recovered safely (T — Tampering). |
| recover() → panicValue.(error) | Library-internal bugs surface via panic; consumer recovery code MUST be able to discriminate via `errors.Is` (D — Denial of service if recovery is impossible). |
