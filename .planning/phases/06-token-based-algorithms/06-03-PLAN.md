    `func PartialRatioScore(a, b string) float64` body structure (Claude's discretion; follows RESEARCH.md Example 4 verbatim in structure):
    1. `if a == b { return 1.0 }` (identity short-circuit BEFORE any byte slicing)
    2. `if len(a) == 0 && len(b) == 0 { return 1.0 }` (both-empty — already covered by the identity check, but keep the guard for clarity per Phase 5 convention)
    3. `if len(a) == 0 || len(b) == 0 { return 0.0 }` (one-empty)
    4. Determine `shorter` / `longer` (`[]byte`) by length comparison: `if len(a) <= len(b) { shorter, longer = []byte(a), []byte(b) } else { shorter, longer = []byte(b), []byte(a) }`
    5. `m, n := len(shorter), len(longer)`
    6. Build `var charSet [256]bool; for _, ch := range shorter { charSet[ch] = true }`
    7. `best := 0.0`
    8. **Region 1 (left tail):** `for i := 1; i < m; i++ { if !charSet[longer[i-1]] { continue }; if r := indelRatio(shorter, longer[:i]); r > best { best = r } }`
    9. **Region 2 (middle):** `for i := 0; i <= n-m; i++ { if !charSet[longer[i+m-1]] { continue }; if r := indelRatio(shorter, longer[i:i+m]); r > best { best = r }; if best == 1.0 { return 1.0 } /* early exit on perfect match */ }`
    10. **Region 3 (right tail):** `if n > m { for i := n - m; i < n; i++ { if !charSet[longer[i]] { continue }; if r := indelRatio(shorter, longer[i:]); r > best { best = r } } }` (the `n > m` guard avoids re-iterating Region 2 when `n == m`)
    11. `return best`

    `func PartialRatioScoreRunes(a, b string) float64` body structure (mirrors the byte path with rune slices and `map[rune]struct{}` charSet):

    2. `if len(a) == 0 && len(b) == 0 { return 1.0 }`
