NBS Tech Note 943 encoding rules (RESEARCH.md §1.4):
1. Delete all vowels unless the vowel begins the word.
2. Remove the second consonant of any double consonants.
3. Reduce codex to 6 letters by joining the first 3 and last 3 letters only.

Comparison rules (6 steps):
1. Reject if |len(codex1) - len(codex2)| >= 3 → return (false, 0).
2. Determine min threshold from Table A using sum of codex lengths.
3. Process L→R, remove identical chars from both.
4. Process R→L on remaining, remove identical chars.
5. similarity = 6 - max(unmatched_count_longer, unmatched_count_shorter).
6. Match iff similarity >= min_threshold.

Threshold Table A:
sum ≤ 4:    threshold 5
4 < sum ≤ 7: threshold 4
7 < sum ≤ 11: threshold 3
sum = 12:   threshold 2
sum > 12:   threshold 2 (clamp — RESEARCH.md Pitfall 7.C, often omitted in summaries)
