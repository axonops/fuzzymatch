<objective>
Land NYSIIS — original Taft-1970 algorithm with 6-character truncation per CONTEXT.md §2 LOCKED. Implementation cites Taft 1970 as algorithmic origin AND Knuth TAOCP Vol. 3 §6.4 as canonical algorithm description (because Taft 1970 is a NY State Special Report unavailable through academic publishers). 80-120 LOC byte-scan + 9-step rule application + final 6-char truncation. Cross-validation against jellyfish==1.2.1 with the LOAD-BEARING `variant_divergence` mechanism — jellyfish ships the MODIFIED/extended NYSIIS variant (no truncation), so ~40-60% of NYSIIS corpus entries (`Catherine → CATARAN` (7 chars), `montgomery → MANTGANARY` (10 chars)) carry `variant_divergence: true` per RESEARCH.md key finding 3 and the loader asserts against the Taft-1970-truncated value. `permittedMongeElkanInner` mutated 16 → 17 entries; `monge_elkan_test.go` rejected slice shrunk 7 → 6 in lockstep. Closes PHON-03.

Purpose: NYSIIS is procedurally simple (9-step Wikipedia/Taft rule sequence transcribes naturally to if-else blocks) but its load-bearing risk is the variant choice — modified-NYSIIS untruncated vs original-Taft-1970-6-char-truncated. The `variant_divergence` mechanism (defensive insurance for Soundex per 07-01; load-bearing here) is exercised in earnest. The truncation property test `PropNYSIIS_CodeLength` and the truncation gate unit test (`len(NYSIISCode("Catherine")) == 6`) lock the variant choice directly. Per Pitfall 7.B (RESEARCH.md §7), an implementer who reads Wikipedia first will easily ship modified-NYSIIS by accident; the property test catches that.

Output: 1 new ~100-LOC algorithm file + 3 companion test files + 1 dispatch wiring file; updates to scripts/gen-phonetic-cross-validation.py (NYSIIS section activated with hand-curated variant_divergence overrides); appended entries to vectors.json (20 NYSIIS entries with divergence tags) + phonetic-codes.json (NYSIIS section) + _staging/nysiis.json; new BDD feature; appended step registrations; appended Five-invariant + PropNYSIIS_CodeLength + PropNYSIIS_CodeCharset; 2 new ExampleXxx; monge_elkan.go + monge_elkan_test.go mutated 16→17 / 7→6 in lockstep + TestMongeElkanScore_BinaryInner_NYSIIS; llms.txt + llms-full.txt synced in-plan.
</objective>
